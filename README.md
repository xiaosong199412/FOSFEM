# FOSFEM
A novel Julia-based GPU-CPU parallelized fundamental package for smoothed finite element method (SFEM), FOSFEM, which includes 4 solvers---FOSFEM-NS, FOSFEM-ES, FOSFEM-FS, FOSFEM-CS using node-based smoothed domain, edge-based smoothed domain, face-based smoothed domain, and cell-based smoothed domain respectively.
